## An generator for yeoman

Uses:

* express
* less
* nunjucks or jade for templating
* requirejs
* jshint
* grunt for precompiling/minification
* Font-Awesome for UI elements (optional)

## Installation

```
npm install -g yo
npm install -g generator-exp
```

## Usage

```
mkdir my-app-name
cd my-app-name
yo exp
```
